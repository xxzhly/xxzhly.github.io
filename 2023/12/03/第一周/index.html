x星期二

一.jdk的安装

1.于物理机右击此电脑，点击属性，找到高级系统设置，选择高级，进入环境变量。

2.于系统变量中新建变量，输入变量名JAVA_HOME，变量值根据自己的jdk的安装路径填写为JDK的bin目录所在路径。

3.在系统变量里新建”classpath”变量，值为：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar; 找到path变量添加变量值：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin。

4.Window+R输入cmd，打开命令行，输入命令java -version，如下图所示则配置成功。

二.Python环境搭建：

1.输入python官网地址，进入官网点击Downloads选择Windows，进入此页面，找到3.6.4版本集，下载安装包。

2.运行安装包，勾选左下角Add python 3.6 to PATH，然后选择第二个自定义安装（Customize installation），其他选项默认，自定义安装路径，完成安装。

3.Windows+R然后输入cmd，打开命令行输入python，如下图所示安装完成

三.永恒之黑漏洞（1）

1.在电脑上安装VMware，并设立好相应的虚拟机系统，win10和kali虚拟机，以及相关文件的下载。

2.关闭win10系统的防火墙，以确保后续操作的进行。

3.获取Win10虚拟机的IPv4地址，“win+R”键打开命令行，输入“cmd”，按下回车，待命令行窗口出现后，输入“ipconfig”命令，查看IPv4地址，并记录下来。

4.kali虚拟机相关文件的准备
（1）.在kali虚拟机上，使用pip install smbclient命令安装python的SMB客户端库，用于与SMB服务器进行通信。等待安装完成。
（2）.下载并解压CVE-2020-0796-PoC-master，将其移至kali虚拟机中。该工具包含了针对CVE-2020-0796漏洞的攻击代码。

5.检验漏洞是否存在，在命令行窗口中输入CVE-2020-0796-Scanner.exe路径来检测设备是否存在CVE-2020-0796漏洞。当确认系统存在此漏洞，便可以进行下一步操作。

6.执行攻击步骤
（1）在kali虚拟机中打开CVE-2020-0796-PoC-master文件夹。
（2）右键点击“以root身份打开”，再右键点击“在这里打开终端”。
（3）在终端中输入“python3 CVE-2020-0796.py win10的ipv4地址”，其中“win10的ipv4地址”为之前记录下的win10虚拟机的IPv4地址。
（4）按下回车键执行攻击命令。观察win10虚拟机的运行结果，预期结果是出现蓝屏现象，结果如下图：

2.将压缩包CVE-2020-0796-Scanner粘贴至Win10。window+R打开一个命令行于命令行输入“ipconfig”查看Win10的ip地址192.168.138.129。,，解压并打开文件夹CVE-2020-0796-Scanner于路径栏输入“cmd”打开命令行。再输入“CVE-2020-0796-Scanner.exe”运行检测工具，输入靶机ip，查看靶机漏洞

6.于桌面打开终端输入以下命令使kali进入监听模式：msfconsole

二、简易钓鱼网站

1.于物理机打开代理，Windows+R输入cmd打开命令行，输入命令ipconfig查询物理机ip地址

2.打开kali，于桌面打开终端，输入sudo su，输入kali密码进入管理员模式

3.输入命令 vim /etc/proxychains4.conf编辑代理设置，下拉至最后一行，单机空处，按i键进入编辑模式，更改ip地址为物理机地址+代理端口的格式，ESC，输入：wq退出并保存代理设置。

4.继续输入proxychains4 git clone https://github.com/makdosx/mip22，下载mip22文件![image-20231202171737784](https://cdn.jsdelivr.net/gh/xoxoxooxox/image@main//imgimgimage-20231202171737784.png)

5.输入命令“chmod -R 777 mip22”复制mip22，“cd mip22”传送到mip22的位置，改写mip22文件的权限

5.输入命令“chmod -R 777 mip22”复制mip22，“cd mip22”传送到mip22的位置，改写mip22文件的权限

6.找到mip22文件夹于空白处右键打开终端，输入命令“proxychains4 bash mip22.sh”，运行程序打开，根据提示数字，创建相应的钓鱼网站
